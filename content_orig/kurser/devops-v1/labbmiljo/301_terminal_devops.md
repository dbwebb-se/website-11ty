Terminal för devops
==================================

Vi behöver så klart en terminal i denna kursen också. Men för er som sitter på Windows och använder Cygwin, så brukar det alltid vara problem för några att installera paketen som behövs. Därför rekommenderar vi att använda [Bash i Windows med WSL](kunskap/installera-bash-i-windows-med-wsl-och-ubuntu) istället.

För Mac och Linux användare kan ni fortsätta använda vad ni gjorde förut.
